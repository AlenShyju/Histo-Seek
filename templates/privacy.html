Privacy Statement

How we protect your data and ensure transparency

Last updated: April 23, 2025

Our Commitment

At Browser History Analyzer, we prioritize your privacy and security above all else. As cybersecurity professionals, we've built this tool with the same privacy standards we demand in our own work.

Core Principle: Your browsing history never leaves your control unless you explicitly choose to share it with third-party services for threat analysis.

Data Handling

Data Collection

We only process the specific browser history file you choose to upload (typically Chrome's "History" file or Firefox's "places.sqlite"). The tool does not:

  • Access other files on your device
  • Collect personal identifiers like names or email addresses
  • Use cookies or tracking technologies
  • Record your IP address beyond standard server logs

Processing

All analysis occurs in a secure, isolated environment:

  • Local processing: The tool runs entirely in your browser for most operations
  • Server processing: When server-side analysis is required, files are encrypted in transit
  • Automatic deletion: Uploaded files are permanently deleted immediately after processing completes
  • Memory-only: No history data is ever written to persistent storage

Data Retention

We maintain strict data retention policies:

  • Session data: Analysis results are kept temporarily in memory and cleared when you close your browser
  • Server logs: Standard web server logs are rotated weekly and contain no history data
  • Backups: No backups containing user data are maintained

Third-Party Services

Certain features integrate with external services to enhance functionality:

VirusTotal API

When you enable suspicious URL detection:

  • Domain names (not full URLs) may be sent to VirusTotal for threat intelligence
  • This is optional and disabled by default
  • VirusTotal's privacy policy applies to any data shared with them

IP Geolocation (ip-api.com)

For geolocation features:

  • Public IP addresses associated with domains may be queried
  • Only approximate location (city-level) is retrieved
  • No personal browsing data is shared with the service

Important: We don't control these third-party services. We recommend reviewing their privacy policies before enabling these features.

Security Measures

We implement multiple layers of protection:

Technical Safeguards

  • HTTPS encryption for all data transfers
  • Memory isolation between user sessions
  • Regular security audits of our codebase
  • Automatic security headers to prevent common web vulnerabilities

Open Source Transparency

As an open-source tool:

  • All code is publicly available for review
  • Security researchers can audit the implementation
  • Users can verify there are no hidden data collection mechanisms
  • Community contributions help strengthen security

Legal Compliance

Data Sharing

We do not share, sell, or disclose your data except:

  • When required by valid legal process
  • To protect our rights or safety
  • When you explicitly opt-in to share data (e.g., submitting a bug report)

Your Rights

You maintain control over your data:

  • Right to decline processing by closing the tool
  • Right to verify no data is retained after your session
  • Right to request deletion of any accidental server logs (via contact form)
