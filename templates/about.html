Deep Forensic Analysis

Comprehensive examination of browser history with advanced recovery techniques:

Active History Extraction

Extracts active browsing history from Chrome (urls table) and Firefox (moz_places table) with complete metadata.

Deleted History Recovery

Recovers deleted history entries using multiple forensic techniques:

  • Hidden or null-time entries (hidden = 1 or last_visit_time IS NULL)
  • Orphaned visits (unlinked records in visits or moz_historyvisits)
  • SQLite free page analysis (regex-based URL extraction)

SQLite Parsing

Advanced parsing of Chrome's History and Firefox's places.sqlite databases with support for:

  • Compressed databases
  • Corrupted file recovery
  • WAL journal analysis

Download History Analysis

Complete analysis of file download activities across browsers:

Chrome Downloads

Extracts complete download history (downloads table) including:

  • File path and name
  • Download timestamp
  • File size and referrer
  • Download status

Firefox Downloads

Extracts download history via moz_annos including:

  • File name and referrer
  • Download date
  • Source URL

Session Data Analysis

Reconstructs browsing sessions to understand user navigation patterns:

  • Extracts session data (URL, referrer, visit time) from Chrome (visits table) and Firefox (moz_historyvisits table)
  • Identifies complete browsing sessions with timestamps
  • Visualizes navigation paths between sites

Advanced Threat Detection

Comprehensive security scanning to identify potential threats in browsing history:

VirusTotal Integration

Automatically scans URLs against VirusTotal API to detect:

  • Known malicious domains
  • Suspicious activity
  • Phishing attempts
  • Malware distribution sites

Suspicious Keyword Matching

Flags URLs containing user-defined suspicious keywords such as:

  • Login pages ("login", "signin")
  • Financial sites ("bank", "paypal")
  • Security risks ("phishing", "scam")
  • Custom keyword lists

Real-time URL Scanning

Allows manual scanning of specific URLs with detailed threat reports including:

  • Detection ratio from multiple antivirus engines
  • Last analysis date
  • Detailed threat categorization

Geolocation Mapping

Visualize browsing activity geographically to identify patterns and anomalies:

IP Geolocation

Extracts geographical data for URLs including:

  • Country and city
  • Latitude and longitude
  • ISP and organization

Text-based Display

Shows geolocation data directly in the history table:

  • Country flags for quick visual reference
  • City-level precision where available
  • Coordinates for mapping integration

Forensic Analysis Process

The forensic analysis process follows these steps:

  1. Database Acquisition: Secure collection of browser history files
  2. File Verification: SHA-256 checksum validation
  3. Active Record Extraction: Parsing standard database tables
  4. Deleted Data Recovery: Advanced forensic techniques
  5. Threat Analysis: VirusTotal integration
  6. Report Generation: Comprehensive forensic documentation
